# راهنمای جامع معماری کدگذاری پویا و شناسه انبار کالا (SKU Engine) - نسخه عددی-محور

این سند مرجع رسمی و استراتژیک سیستم کدگذاری انبار (SKU - Stock Keeping Unit) برای تمامی محصولات توزیع‌شده تحت بستر هلدینگ ARIA-G در فروشگاه‌ساز شاپفا است. این سیستم به گونه‌ای مهندسی شده است که با توسعه کاتالوگ کالایی در آینده (افزودن دسته‌ها، برندها و زیردسته‌های جدید و برندهای متفرقه)، هیچ‌گونه تداخل یا آسیبی به کدهای قدیمی وارد نکرده و به راحتی توسعه یابد.

در این نسخه، سهم حروف انگلیسی به حداقل رسیده و سیستم به صورت کاملاً عددی-محور (Numeric-Dominant) با قابلیت توسعه بی‌پایان بازطراحی شده است.

---

## ۱. ساختار کدگذاری سگمنتی عددی (Segmented Numeric SKU Architecture)

هر کد SKU در این سیستم متشکل از ۴ سگمنت عددی مشخص است که با کاراکتر خط تیره (`-`) از یکدیگر تفکیک می‌شوند:

$$\mathbf{SKU} = \mathbf{BRN} - \mathbf{CAT} - \mathbf{MDL} - \mathbf{VAR}$$

### تشریح سگمنت‌های عددی ۴ گانه:

```
[BRN] - [CAT] - [MDL] - [VAR]
  │       │       │       └── تنوع رنگ/سایز/وزن (۳ رقم)
  │       │       └── کد سری/مدل کالا (۴ رقم)
  │       └── کد دسته و زیردسته کالا (۳ رقم)
  └── کد برند محصول (۳ رقم)
```

1.  **سگمنت ۱: برند محصول (BRN - Brand Code - ۳ رقم):**
    *   **کدهای ۱۰۰ تا ۸۹۹:** به برندهای اصلی و ثبت‌شده اختصاص می‌یابند.
        *   *مثال:* سی‌کلاس = `110`، فابرکاستل = `120`، زبرا = `130`.
    *   **کدهای ۹۰۰ تا ۹۹۹:** به **برندهای متفرقه (Miscellaneous Brands)** اختصاص می‌یابند.
        *   *مثال:* برندهای متفرقه ایرانی = `900`، برندهای متفرقه وارداتی/چینی = `990`، برندهای متفرقه عمومی = `999`.

2.  **سگمنت ۲: کد دسته و زیردسته کالا (CAT - Category Code - ۳ رقم):**
    *   **رقم اول (دسته اصلی):**
        *   `1` = ابزارهای نگارش (Writing)
        *   `2` = ابزارهای اصلاح و پاک‌کننده (Correctors)
        *   `3` = ملزومات چسباندن (Adhesives)
        *   `4` = ابزارهای تراش و جانبی (Sharpeners)
        *   `5` = لوازم هنری و رنگ‌آمیزی (Art & Coloring)
    *   **رقم دوم و سوم (زیردسته تخصصی):**
        *   در دسته نگارش (`1`): روان‌نویس ژله‌ای = `10`، خودکار معمولی = `20`، اتود = `30`، مداد چوبی = `40`. (کد کل: `110` برای روان‌نویس ژله‌ای)
        *   در دسته اصلاح (`2`): پاک‌کن جامد = `10`، لاک غلط‌گیر = `20`. (کد کل: `210` برای پاک‌کن)
        *   در دسته چسب‌ها (`3`): چسب ماتیکی = `10`، چسب مایع = `20`. (کد کل: `310` برای چسب ماتیکی)
        *   در دسته تراش‌ها (`4`): تراش اتود = `10`، تراش مداد = `20`. (کد کل: `410` برای تراش اتود)

3.  **سگمنت ۳: مدل و سری کارخانه (MDL - Model Code - ۴ رقم):**
    کد عددی تعیین‌کننده مدل فیزیکی کالا.
    *   *مثال:* کاندید = `0500`، مورنو = `0700`، مدل ER-60 = `0060`، مدل GS-AP36 = `0036`، ژوپیتر = `0200`، مدل AR1412 = `1412`.

4.  **سگمنت ۴: تنوع و متغیر (VAR - Variant Code - ۳ رقم):**
    *   **کد `000`:** کالای ساده / تک‌مدل / فاقد متغیر مستقل (Single/Default).
    *   **کدهای کلاسبندی رنگ‌ها:**
        *   آبی = `101`، مشکی = `102`، قرمز = `103`، سبز = `104`، صورتی = `105`، بنفش = `106`.
    *   **کدهای کلاسبندی مقادیر وزنی/حجمی:**
        *   ۳۶ گرم = `036`، ۱۲ میلی‌لیتر = `012`، ۸ گرم = `008`.

## ۳. توجیه پشتیبانی از «برندهای متفرقه» (Miscellaneous Brands)

اگر در آینده کالایی متفرقه تحت عنوان **«پاک‌کن فانتزی متفرقه وارداتی مدل فضانورد»** به کاتالوگ انبار اضافه شود، تخصیص شناسه عددی آن به صورت زیر انجام می‌گیرد:
*   کد برند متفرقه وارداتی: `990`
*   کد دسته پاک‌کن: `210`
*   کد مدل فضانورد: `9101` (استفاده از رده ۹۰۰۰ برای مدل‌های متفرقه)
*   کد تنوع ساده: `000`
*   **شناسه SKU نهایی:** `990-210-9101-000`

این شناسه بدون هیچ‌گونه تداخل با کاتالوگ سی‌کلاس (که کدهای برندشان با `110` آغاز می‌شود)، به صورت موازی در سیستم انبارداری شاپفا ثبت می‌گردد.
