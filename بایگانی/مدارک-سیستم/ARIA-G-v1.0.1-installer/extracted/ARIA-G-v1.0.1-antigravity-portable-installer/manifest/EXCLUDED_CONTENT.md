# Excluded Content List
The following files and directories are excluded from the portable installer package to maintain package purity:
* **tests/** and **tests/runs/** - Excluded synthetic fixtures, traces, results, logs, and schemas.
* **private-local/** - Excluded fictional database registries, schemas, campaigns, and mock entities.
* **deployment/** - Excluded build and readiness files.
* **release/** - Excluded build staging directories and files.
* **Historical Reports** - Excluded original reviews, correction reports, test reports, and defect reports.
